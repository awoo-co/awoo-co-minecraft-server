Welcome to awoo-co Minecraft Server Beta

The Minecraft mods and Tailscale readme file is included in the zip.

The server IP is also in the readme file within the zip.

Instructions:

  1. First, download the zip file and extract it.
  2. Then, download the CurseForge launcher from the CurseForge website.
  3. Open the CurseForge launcher and click on Minecraft.
  4. Click on "Import Profile" or just "Import".
  5. Find awoo-co.zip and select it.
